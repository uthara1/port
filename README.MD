# Personal Portfolio Website

This repository contains the clean, responsive codebase for my academic and professional development portfolio. It acts as an interactive overview of my technical competency, ongoing project milestones, and hands-on developer experience.

## 🛠️ Built With
- **HTML5**: Structured semantic layout documentation.
- **CSS3**: Custom element properties, structural grid/flexbox sizing patterns, dynamic viewport handling.
- **FontAwesome**: Scalable vector icons for direct interface links.

## 📂 Project Repository Setup Instructions
1. Clone this repository locally.
